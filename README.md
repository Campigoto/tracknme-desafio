
## Requisitos
- java 11+
 - springboot
 - junit5
 - h2 database

## Objetivo
- Sistema usado para gerenciar os funcionários de uma empresa pequena.

## Uso

Para rodar o projeto execute:
```./mvnw spring-boot:run```


- O servidor irá iniciar em localhost na porta 8080 por padrão
